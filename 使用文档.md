# 电子人情簿使用文档

本说明面向日常使用，不需要懂代码。

---

## 1. 快速开始

## 方式 A：双击一键启动（推荐）

1. 进入项目目录。
2. 双击：`一键打开人情簿.bat`
3. 浏览器会自动打开网页（默认地址：`http://localhost:4173`）。

## 方式 B：命令行启动

在项目目录执行：

```bash
npm run dev
```

然后打开终端提示的本地地址（常见是 `http://localhost:5173`）。

---

## 2. 页面功能说明

## 顶部区域

- **主题标题**：可输入本次人情簿主题（例如“2026 春节人情簿”）。
- 标题会自动保存，下次打开仍保留。

## 左侧区域

- **姓名**：必填
- **礼金（元）**：必填，数字格式
- **备忘录**：选填

操作按钮：

- **添加 / 保存修改**
- **清空表单 / 取消修改**
- **导出 PDF**
- **导出 Excel**
- **导入日志数据**

## 右侧区域

- 展示当前页记录
- 显示：
  - 总条数
  - 当前页 / 总页数
  - 本页礼金总额（阿拉伯数字）
  - 全部礼金总额（阿拉伯数字）
- 分页按钮：首页 / 上一页 / 下一页 / 末页

---

## 3. 基本操作

## 3.1 新增记录

1. 在左侧输入姓名、礼金、备忘录。
2. 点击 **添加**。
3. 记录会出现在右侧列表中。

## 3.2 修改记录

1. 在右侧目标记录点击 **修改**。
2. 左侧表单会自动填入原数据。
3. 修改后点击 **保存修改**。

## 3.3 删除记录

1. 在右侧点击 **删除**。
2. 在确认框点击“确定”后删除。

---

## 4. 导出功能

## 4.1 导出 PDF

- 点击左侧 **导出 PDF**。
- 导出的文件名会使用当前主题标题。
- PDF 中包含标题和数据表格。

## 4.2 导出 Excel

- 点击左侧 **导出 Excel**。
- 导出的文件名会使用当前主题标题。
- 每页数据会生成一个工作表。
- 表中包含：
  - 姓名
  - 礼金（大写）
  - 礼金（数字）
  - 备忘录
  - 添加时间

---

## 5. 日志与导入

系统会记录操作日志（新增、修改、删除、标题修改），写入：

- `data/operation-logs.json`

## 从日志恢复数据

点击左侧 **导入日志数据**，系统会：

1. 读取日志文件；
2. 回放操作历史；
3. 恢复当前记录和主题标题。

注意：导入会覆盖当前页面记录。

---

## 6. 常见问题

## 问题 1：双击脚本后打不开网页

请手动尝试打开：

- `http://localhost:4173`
- `http://localhost:5173`

如果仍打不开：

1. 关闭旧的命令行窗口；
2. 重新双击 `一键打开人情簿.bat`。

## 问题 2：日志文件没更新

确认 API 已启动：

- `http://localhost:3001/api/health`

能看到 `{"ok":true}` 说明正常。

## 问题 3：导出失败

1. 先刷新网页后重试；
2. 确认服务正在运行；
3. 再尝试导出。

---

## 7. 使用建议

- 每次录入完及时导出一份 Excel 备份。
- 大规模修改前先导出一次备份。
- 可定期复制 `data/operation-logs.json` 做外部备份。
